Beweisverfahren

Logische Beweisverfahren sind methodische Ansätze, um die Gültigkeit oder Widerlegbarkeit von Aussagen in einer formalen Sprache zu prüfen.
Sie sind in der Logik zentral, da sie es ermöglichen systematisch zu entscheiden, ob eine gegebene Aussage aus einer Menge von Prämissen folgt.

Ein logischer Beweis ist eine Folge von Schritten, die durch logische Regeln definiert ist, um eine Aussage (den Satz) aus den gegebenen Prämissen abzuleiten. Ein Beweis zeigt, dass die Aussage in allen Modellen gilt, in denen die Prämissen wahr sind.

Arten von Beweisen

  • Direkter Beweis

    Man zeigt, dass eine Annahme aus den Prämissen folgt.

  • Indirekter Beweis (Widerspruchsbeweis)

    Man nimmt die negation der Annahme an, und zeigt, dass sie aus den Prämissen zu einem Widerspruch führt.

  • Vollständige Induktion

    Man zeigt, dass eine Aussage für alle natürlichen Zahlen gilt.

    1. Induktionsanfang: Es wird gezeigt, dass eine Aussage für einen bestimmten Anfangswert (meist 0 oder 1) wahr ist.
    2. Induktionsschritt:
      • Induktionsannahme: Man nimmt an, dass die Aussage für eine beliebige, aber feste natürliche Zahl n gilt.
      • Induktionsschluss: Unter Verwendung der Induktionsannahme beweist man, dass die Aussage dann auch für die folgende Zahl n+1 wahr ist.
