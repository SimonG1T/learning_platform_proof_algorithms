Das DPLL-Verfahren

Das DPLL-Verfahren ist das wichtigste Verfahren zur Entschedung des allgemeinen Erfüllbarkeitsproblems (SAT-Problem). Es wurde 1961 von Davis Logemann und Loveland eingeführt und ist eine Weiterentwicklung desvon Davis und Putnam entwickelten DP-Verfahren von 1960.

  1. Logische Grundlagen

    • Klauseln: Eine Klausel ist eine Disjunktion (logisches Oder) von Literalen, wobei ein Literal eine atomare Aussage oder deren Negation ist.
    • Konjunktive Normalform (KNF): Jede logische Formel muss in eine konjunktive Normalform umgewandelt werden, d.h., sie muss als Konjunktion (logisches Und) von Klauseln vorliegen.

  2. Reduzierung von Literalen

    Aus einer Menge von Klauseln, wird ein Literal ausgewählt. Anschließend werden alle Klauseln gestrichen, welche das Literal enthalten. Aus den übrigen Klauseln, wird das Komplement (die Negation des Literals) gestrichen.

  3. Verfahren

    • Klauselumformung: Die gesamte Formel wird in konjunktive Normalform umgewandelt.
    • Jedes Modell M von F, welches L zu wahr auswertet, ist Modell von F_L. Zu jedem Modell M' von F_L gibt es ein Modell M von F, das Atome in F_L so auwertet wie M': M(L)=1, M wie M' sonst. Also ist F_L erfüllbar, g.d.w. F ein Modell hat, das L zu wahr auswertet.
      Schritte

      Sei F Menge von Klauseln.

    1. F={}: F erfüllbar
    2. □ in F: F ist unerfüllbar
    3. Einerklausel {L} in F: F erfüllbar gdw. F_L erfüllbar
    4. L Literal in F: F erfüllbar gdw. F_L erfüllbar oder F_!L erfüllbar
