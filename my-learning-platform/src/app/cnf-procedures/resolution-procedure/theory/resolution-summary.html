Das Resolutionsverfahren

Das Resolutionsverfahren ist ein logisches Schlussverfahren, das häufig in der automatischen Theorembeweisung verwendet wird, insbesondere in der Prädikatenlogik erster Stufe. Es basiert auf der Technik der Auflösung (Resolution), die zur Ableitung eines Widerspruchs dient und somit zur Überprüfung der Gültigkeit eines logischen Ausdrucks beiträgt. Das Verfahren wurde von John Alan Robinson in den 1960er Jahren entwickelt.

  1. Logische Grundlagen

    • Klauseln: Eine Klausel ist eine Disjunktion (logisches Oder) von Literalen, wobei ein Literal eine atomare Aussage oder deren Negation ist.
    • Konjunktive Normalform (KNF): Jede logische Formel muss in eine konjunktive Normalform umgewandelt werden, d.h., sie muss als Konjunktion (logisches Und) von Klauseln vorliegen.

  2. Resolutionsregel

    • Die Resolutionsregel besagt, dass aus zwei Klauseln, die ein komplementäres Literal enthalten (ein Literal und dessen Negation), eine neue Klausel abgeleitet werden kann, indem das komplementäre Paar eliminiert wird.
    • Formal: Aus A|L&A|L und B|¬L&B|¬L wird die neue Klausel A|B&A|B gebildet, wobei A|A und B|B jeweils Disjunktionen von Literalen sind und LL ein Literal ist.

  3. Verfahren

    • Negation des Beweises: Um zu beweisen, dass eine Aussage C aus einer Menge von Aussagen S folgt (S⊨C), fügt man die Negation der Aussage C zur Menge S hinzu.
    • Klauselumformung: Die gesamte Formel wird in konjunktive Normalform umgewandelt.
    • Iterative Anwendung der Resolutionsregel: Wiederholt wird die Resolutionsregel auf die Klauselmenge angewendet, um neue Klauseln zu erzeugen. Dies wird fortgesetzt, bis entweder die leere Klausel (Widerspruch) erzeugt wird oder keine neuen Klauseln mehr gebildet werden können.
    • Ergebnisinterpretation: Wenn die leere Klausel erzeugt wird, bedeutet dies, dass ein Widerspruch vorliegt, was beweist, dass CC eine logische Konsequenz von S ist. Andernfalls kann nicht geschlossen werden, dass C aus S folgt.
