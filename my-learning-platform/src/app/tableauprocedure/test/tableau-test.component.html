Teste das Tableauverfahren

Das Tableauverfahren generiert mithilfe von Formeln einen Baum.
Dies geschieht so lange, bis alle Knoten verwendet wurden.

Mögliche Eingabe:

  • Variablen als einzelne Klein- oder Großbuchstaben: A,B,C a,b,c
  • Klammern werden als begrenzer verwendet: '(' ')'
  • Logische Zeichen werden wie folgt ausgewählt:
    • NOT: als '!'
    • AND: als '&'
    • OR: als '|'
    • Implication: als '->'
    • Equivalence: als '<->'

Mögliche Tests zur Schnellauswahl:

  • (A|B)&(!A|C)&(!B)
  • (A|B)&(!A|B)&(A|!B)
  • (!(!(A|B)|A))
  • (!(!(A&B)&A))
